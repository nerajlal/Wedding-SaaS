1. Product Overview
Velvet Vows is an ultra-premium digital wedding invitation platform that empowers couples to craft, personalise, and share stunning digital invitations — with zero technical expertise required. The product sits at the intersection of elegance, simplicity, and modern web technology.
1.1 Vision Statement
⬥ Core Vision
To become the most beautiful and effortless way for couples to announce their love story — delivering the luxury feel of a physical invitation through the immediacy and reach of a digital experience.
1.2 Mission
Velvet Vows democratises premium wedding stationery. Where a printed luxury invitation suite may cost thousands, Velvet Vows delivers equivalent aesthetic prestige — shareable in seconds, accessible on any device, memorable forever.

9. Security & Privacy
Google OAuth only — no passwords stored by Velvet Vows
Session tokens stored in HTTP-only, Secure, SameSite=Strict cookies
All API routes protected by 
Laravel auth middleware
Invitation ownership enforced — users can only edit their own invitations
Public invitation pages are intentionally unauthenticated (by design)
Payment security:
Razorpay used for secure transactions
Payment verification via 
signature validation + webhooks
No sensitive payment data stored on server
Photo uploads validated server-side (file type, size, MIME)
Rate limiting on public view counter endpoint (IP-based, 1 req/10s)
HTTPS enforced via Vercel's automatic SSL
GDPR note: User data limited to name, email from Google; deletable on request

13. Glossary
Term
Definition
Slug
A short, URL-safe unique identifier for each invitation (e.g. 'rahul-priya-2026')
Session
A server-side record linking a browser to an authenticated user, stored via HTTP-only cookie
OAuth 2.0
Open standard for access delegation — used here so users can sign in with Google without sharing their password with Velvet Vows
QR Code
Quick Response code — a machine-readable 2D barcode that encodes the invitation URL
Nanoid
A tiny, secure URL-friendly unique string ID generator used to create invitation slugs
SSR
Server-Side Rendering — feature that renders pages on the server for performance and SEO
CTA
Call To Action — A button or element prompting a user to take a specific action
RSVP
Répondez s'il vous plaît — a request for a response to a wedding invitation
